# 🎮 Random Pair Generator

[![Play Now](https://img.shields.io/badge/Play%20Now-Live%20Demo-brightgreen?style=for-the-badge)](https://p-iggsray.github.io/Random-Pair-Generator/)

A quick little random team generator that pairs inexperienced players with experienced ones for **fair matchmaking** and easy tournament bracket creation!

## How It Works

1. **Add your players** – Split them into Experienced and Inexperienced categories
2. **Generate teams** – Hit the button and watch the magic happen ✨
3. **Export results** – Copy your pairings straight to Challonge (or wherever!)

## Features

- ⚡ Instant random pairing with balanced skill levels
- 📋 Export-ready tournament format
- 💾 Auto-saves your lists (no data lost!)
- 🌙 Sleek dark theme UI
- 📱 Mobile-friendly design

**That's it!** Start pairing. 🚀
